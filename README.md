# 사상체질 진단 웹 (干支體質論) — 배포 안내서

신라한의원 · 황교헌 원장 전용

## 📁 이 폴더 구조

```
sasang-web/
├── api/
│   └── analyze.js        ← 체질감별 엔진 (서버에서만 실행 · IP 은닉)
├── public/
│   ├── index.html        ← 사용자가 보는 화면 (입력·결과·섭생법)
│   └── guidance-data.js  ← 섭생법 데이터 (권도원 8체질, 공개 출처)
├── vercel.json           ← Vercel 설정
├── package.json          ← Node 런타임 설정
└── README.md             ← 이 파일
```

### 보안 구조 (중요)
- **`api/analyze.js`** = 干支體質論 알고리즘 전부. **서버에서만 실행**되고 브라우저로 내려가지 않습니다. 사용자가 F12로 코드를 열어봐도 절대 보이지 않습니다.
- **`public/` 안의 파일** = 누구나 볼 수 있는 화면 코드. **계산식은 한 줄도 없습니다.** 생년월일을 서버에 보내고, 결과만 받아 표시합니다.
- 추후 창약·결제 기능도 `api/` 폴더에 추가하면 같은 방식으로 보호됩니다.

## 🚀 배포 방법 (가장 쉬운 길 — 명령어 불필요)

### 1단계. Vercel 가입
1. https://vercel.com 접속 → **Sign Up**
2. **Continue with GitHub** (또는 이메일) — 무료 플랜으로 충분

### 2단계. 이 폴더 업로드
**방법 A — 드래그앤드롭 (가장 간단)**
1. Vercel 로그인 후 대시보드에서 **Add New... → Project**
2. **Deploy a folder** / 또는 이 `sasang-web` 폴더를 통째로 끌어다 놓기
3. 자동으로 `api/`를 서버 함수로, `public/`을 웹페이지로 인식합니다
4. **Deploy** 클릭 → 1~2분 후 `https://sasang-web-xxxx.vercel.app` 주소 생성

**방법 B — GitHub 연동 (수정 시 자동 반영, 추천)**
1. GitHub에 이 폴더를 새 저장소로 올림
2. Vercel 대시보드 → **Add New → Project → Import** → 그 저장소 선택
3. 설정 변경 없이 **Deploy**
4. 이후 GitHub에 파일을 고치면 자동으로 사이트가 갱신됩니다

### 3단계. 확인
- 생성된 주소 접속 → 생년월일 선택 → **체질 감별하기**
- 결과·간지·섭생법이 나오면 성공

### 4단계. (선택) 내 도메인 연결
- Vercel 프로젝트 → **Settings → Domains** → 보유 도메인 입력
- 신라한의원 도메인이 있다면 `체질.신라한의원.com` 같은 식으로 연결 가능
